sdb_get_table_column

sdb_get_table_column ( string $table_name )
Description:

sdb_get_table_column () Returns all columns of the selected table

Parameters:

$table_name | Table name from the selected database

Examples for sdb_get_table_column

Example 1: Returns columns of the table

<?php
require_once("db.php");

sdb_init();

sdb_open_db("database_name");

sdb_query("CREATE TABLE table_name (id PKEY, name VARCHAR(99))");
// Create the table

sdb_get_table_column("table_name");
?>
