sdb_open_db

sdb_open_db ( string $database_name )
Description:

sdb_open_db () One of the major function of SDB API . Attempt to create new database if it does not exists or try to open existing database.

Parameters:

$database_name | Database name which you want to create one or open the existing one.

Examples for sdb_open_db

Example 1: Creating or Opening new database

<?php
require_once("db.php");

sdb_init();

sdb_open_db("database_name");

// SQL Queries will be here

?>
