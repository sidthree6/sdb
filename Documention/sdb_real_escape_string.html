sdb_real_escape_string

sdb_real_escape_string ( string $string )
Description:

sdb_real_escape_string () Remove unwanted charactors from the string.

Parameters:

$string | Input the string

Examples for sdb_real_escape_string

Example 1: Create perfect string for query

<?php
require_once("db.php");

sdb_init();

sdb_open_db("database_name");

$string = "Hello, i'm user";
echo sdb_real_escape_string($string);
//print Hello, i\'m user. This string will protect your database against hacking
?>
