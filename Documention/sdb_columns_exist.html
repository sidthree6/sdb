sdb_column_exist

sdb_column_exist ( string $table_name , string $column_name )
Description:

sdb_column_exist () check if column exists in the selected database and table

Parameters:

$table_name | Table name which is currently exists in the database

$column_name | Valid column name from the selected table

Examples for sdb_column_exist

Example 1: Check if column exists

<?php
require_once("db.php");

sdb_init();

sdb_open_db("db_name");

//Create a table in the database
sdb_query("CREATE TABLE table_name (id PKEY,name VARCHAR(255))");

if(sdb_column_exist("table_name","name"))
{
    echo "Column exist";
    // If column exist then echo this
}
else
{
    echo "Column does not exist";
    // If column does not exist then echo this
}

?>
