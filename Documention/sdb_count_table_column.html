sdb_count_table_column

sdb_count_table_column( string $table_name , string $column_name )
Description:

sdb_count_table_column () count columns in the table

Parameters:

$table_name | Table name which is currently exists in the database

$column_name | Valid column name from the selected table

Examples for sdb_count_table_column

Example 1: Count columns in the table

<?php
require_once("db.php");

sdb_init();

sdb_open_db("db_name"); 

//Create a table in the database 
sdb_query("CREATE TABLE table_name (id PKEY,name VARCHAR(255))");

//Insert data in the table
sdb_query("INSERT INTO table_name (name) VALUES ('first') ");
sdb_query("INSERT INTO table_name (name) VALUES ('second') "); 

// Output 2
echo sdb_count_table_column("table_name","name");

?>
