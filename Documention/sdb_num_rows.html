sdb_num_rows

sdb_num_rows ( string $query )
Description:

sdb_num_rows () Returns the numbers of rows affected by sdb_query

Parameters:

$query | Only select table query works with the sdb_num_rows

Examples for sdb_num_rows

Example 1: Getting number of rows affcted by Query
